Detailed Description

Functions to identify contact points.

Currently, only a simple and specialized "tri foot" collision detection algorithm is included. This means that contact points can only be identified when a body is collided as a cylindrical foot towards a plane terrain.

Collides a body as a cylindrical foot against the terrain.

The body position is considered to be geometrical center of the cylinder. If a collision is detected necessary constraints are added to the Jacobian.

Parameters
mPointer to the memory representing the simulation world.
cyl_radiusRadius of the cylinder.
cyl_heightHeight of the cylinder.
contacts_offsetOffset for the contact constraints rows in the Jacobian matrix.
foot_bodyIndex for body to collide as foot, in the interval [0, TP_BODIES-1].
Returns
Number of constraint rows added.
